# User - Follow

## Description

Follow a user on twitch (and turn of notifications if desired).

## Method Definition

*Required Parameters*

| Name          | Type      | Description  |
| ------------- |:---------:| -----------------:|
| authToken | string | User Auth Token |
| targetUserId | number | Target User Id |
| notification | boolean | Enable notifications? |

*Optional Parameters*

None
