# EventHandler - Simple

## Description

The SimpleEventHandler will process all events synchronously in the current thread.

## Dependencies

<JavaDependency group="com.github.philippheuer.events4j" name="events4j-handler-simple" version="0.12.1" scope="implementation" />

## Set as default in Twitch4J and for eventManager.onEvent

<Tabs groupId="code">
<TabItem value="java" label="Java">

```java
ITwitchClient twitchClient = TwitchClientBuilder.builder()
    .withDefaultEventHandler(SimpleEventHandler.class)
    .build();
```

</TabItem>
