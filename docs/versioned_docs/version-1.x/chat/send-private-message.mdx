# Send Private Message

## Description

Sends a private message to a user via IRC.

:::caution

Twitch [decommissioned](https://discuss.dev.twitch.com/t/deprecation-of-chat-commands-through-irc/40486) chat commands over IRC on February 24, 2023 for third-parties.
Please migrate to [`TwitchHelix#sendWhisper`](https://twitch4j.github.io/javadoc/com/github/twitch4j/helix/TwitchHelix.html#sendWhisper(java.lang.String,java.lang.String,java.lang.String,java.lang.String)).

:::

## Example

<Tabs groupId="code">
<TabItem value="java" label="Java">

```java
twitchClient.getChat().sendPrivateMessage("twitch4j", "Hey! This is a direct message.");
```

</TabItem>
<TabItem value="kotlin" label="Kotlin">

```kotlin
twitchClient.chat.sendPrivateMessage("twitch4j", "Hey! This is a direct message.")
```

</TabItem>
<TabItem value="groovy" label="Groovy">

```groovy
twitchClient.chat.sendPrivateMessage("twitch4j", "Hey! This is a direct message.")
```

</TabItem>
